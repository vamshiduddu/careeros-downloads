🖥️ Desktop App

Interview Copilot — Windows, macOS, Linux
⊞ Windows.exe installer macOS.dmg (Apple Silicon) 🐧 Linux.AppImage 🐧 Linux.deb (Debian/Ubuntu)
  1. macOS — open the .dmg and drag CareerOS into Applications.
  2. Open Terminal and run:
    xattr -cr /Applications/CareerOS.app
  3. Launch CareerOS from Applications. Done.
Why the Terminal step? The macOS build isn't yet Apple-notarized, so macOS may show "CareerOS is damaged and can't be opened". The xattr -cr command clears the quarantine flag — it does not modify the app. Right-click → Open alone will not fix the "damaged" message.

Windows: if SmartScreen warns, click More info → Run anyway.

🧩 Browser Extension

Job-application autofill — Chrome & Firefox
🌐 Chrome / Edge.zip 🦊 Firefox.zip
  1. Download and unzip the file.
  2. Open chrome://extensions (or about:debugging in Firefox).
  3. Enable Developer mode (top-right).
  4. Click Load unpacked and select the unzipped folder.
  5. Pin the CareerOS icon and sign in.
